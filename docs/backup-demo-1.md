# Respaldo Demo 1

Este punto representa la version demo 1 del CRM de Clima Activa.

## Que incluye

- Dashboard interactivo.
- Empresas con crear, editar, filtros y ficha.
- Historial de interacciones por empresa.
- Campanas con destinatarios editables, vista previa y metricas.
- Plantillas con crear, editar, duplicar y activar/desactivar.
- Administracion limpia con futuras integraciones: Gmail, WhatsApp e importacion CSV/Excel.
- Datos demo persistidos en localStorage.
- Esquema inicial Supabase.

## Como volver a esta version con git

```bash
git checkout demo-1
```

Para volver al trabajo normal despues:

```bash
git checkout main
```

## Como volver desde el zip

Descomprimir `backups/climactiva-crm-demo-1.zip` en una carpeta nueva y ejecutar:

```bash
npm install
npm run dev
```
